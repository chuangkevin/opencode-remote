# Archived / Superseded Changes

The two change proposals in this folder describe the **original** architecture:
a Dispatch-style model with PWA frontend + Fastify server + SQLite + local
runner + Web Push notifications.

That entire architecture was **abandoned** in favor of a minimal transparent
HTTP proxy that exposes OpenCode's native web UI. See:

- `docs/superpowers/specs/2026-04-20-redesign-design.md` — redesign spec
- `CLAUDE.md` — AI-facing context for the new architecture
- `openspec/specs/` — current capability specs

Keep these files for historical context. Do not resume work from them.
